# MyChart - Product Requirements Document

## 1. 서비스 개요

### 1.1 비전
모멘텀 투자자를 위한 최적의 종목 발굴 및 모니터링 플랫폼

### 1.2 서비스 정의
MyChart는 한국 주식 시장(KOSPI, KOSDAQ)에서 모멘텀이 있는 주도주를 발굴하고, 개인 관심종목을 효율적으로 관리하며, 차트 분석과 실시간 뉴스를 통해 투자 의사결정을 지원하는 웹 서비스입니다.

### 1.3 핵심 가치
| 가치 | 설명 |
|------|------|
| 주도주 발굴 | 복합 지표 기반으로 시장을 리드하는 종목을 자동으로 선별 |
| 효율적인 관리 | 관심종목을 그룹별로 체계적으로 관리 |
| 심층 분석 | 커스텀 가능한 기술적 지표로 차트 분석 지원 |
| 매매 신호 | 사용자 정의 조건에 따른 매수/매도 신호를 차트에 표시 |
| 정보 집약 | 종목별 관련 뉴스를 한눈에 확인 |

---

## 2. 타겟 사용자

### 2.1 주요 페르소나
**중급 개인 투자자 (모멘텀/추세 투자자)**

- 주식 투자 경험 1년 이상
- 기술적 분석(차트)에 관심이 있음
- 모멘텀/추세 추종 전략 선호
- 거래량, 신고가, 상승률 등 모멘텀 지표에 민감
- 다수의 관심종목을 효율적으로 모니터링하고 싶어함

### 2.2 사용자 니즈
1. **빠른 주도주 발굴**: 시장에서 모멘텀이 발생하는 종목을 신속하게 찾고 싶다
2. **체계적인 관심종목 관리**: 관심있는 종목들을 그룹별로 정리하고 쉽게 접근하고 싶다
3. **심층적인 차트 분석**: 다양한 기술적 지표를 활용해 매매 타이밍을 분석하고 싶다
4. **종목 관련 정보 확인**: 관심 종목의 뉴스와 이슈를 빠르게 파악하고 싶다

## 3. 핵심 기능

### 3.1 주도주 스크리너
시장에서 모멘텀이 발생하고 있는 주도주를 복합 지표 기반으로 자동 선별합니다.

**선정 기준 (복합 지표)**
- 거래량 급증: 5일/20일 평균 대비 거래량 증가율
- 가격 상승률: 일간/주간 상승률
- 신고가 돌파: 52주/연중 신고가 돌파 여부
- 외국인/기관 수급: 순매수 동향

**제공 정보**
- 주도주 리스트 (실시간 업데이트)
- 각 지표별 랭킹
- 테마/섹터별 분류

### 3.2 관심종목 관리
개인화된 관심종목 등록 및 관리 기능을 제공합니다.

**주요 기능**
- 그룹별 관심종목 등록 (예: 성장주, 배당주, 단기매매 등)
- 관심종목 실시간 시세 조회
- 관심종목 순위 변경 및 정렬
- 메모 기능 (매수가, 목표가, 메모 등)

### 3.3 차트 뷰어
캔들차트와 다양한 기술적 지표를 커스텀하여 분석할 수 있습니다.

**차트 유형**
- 캔들차트 (일/주/월봉)
- 거래량 차트

**기술적 지표 (커스텀 가능)**
| 카테고리 | 지표 |
|----------|------|
| 추세 지표 | 이동평균선 (5/10/20/60/120일), MACD |
| 모멘텀 지표 | RSI, 스토캐스틱 |
| 변동성 지표 | 볼린저밴드, ATR |
| 거래량 지표 | OBV, 거래량 이동평균 |

**사용자 설정**
- 지표 ON/OFF 선택
- 지표 파라미터 조정 (기간 등)
- 차트 레이아웃 저장

**매매 신호 시스템**
사용자가 정의한 조건에 따라 차트에 매수/매도 신호를 시각적으로 표시합니다.

| 구분 | 설명 |
|------|------|
| 신호 표시 | 차트 상에 매수(▲) / 매도(▼) 마커로 표시 |
| 조건 설정 | 사용자가 직접 매수/매도 조건 정의 |
| 프리셋 제공 | 자주 사용되는 전략 템플릿 제공 |
| 백테스트 뷰 | 과거 차트에서 신호 발생 지점 확인 |

*조건 설정 예시*
| 전략 유형 | 매수 조건 | 매도 조건 |
|-----------|-----------|-----------|
| RSI 과매도/과매수 | RSI < 30 | RSI > 70 |
| 이평선 크로스 | 5일선이 20일선 상향돌파 | 5일선이 20일선 하향돌파 |
| 볼린저밴드 | 하단밴드 터치 | 상단밴드 터치 |
| MACD 크로스 | MACD 골든크로스 | MACD 데드크로스 |
| 복합 조건 | RSI < 30 AND 거래량 급증 | RSI > 70 AND 음봉 |

*주요 기능*
- 조건 빌더: 드래그 앤 드롭 방식으로 조건 조합
- AND/OR 논리 연산자 지원
- 다중 신호 전략 저장 및 관리
- 신호 발생 시 알림 설정 (선택)

### 3.4 뉴스 피드
종목별 관련 뉴스를 실시간으로 제공합니다.

**주요 기능**
- 종목 관련 뉴스 목록
- 뉴스 키워드 하이라이트
- 관심종목 뉴스 알림
- 뉴스 감정 분석 (긍정/부정/중립)

## 4. 사용자 시나리오

### 시나리오 1: 주도주 발굴 및 분석
```
1. 사용자가 '주도주 스크리너' 페이지에 접속
2. 오늘의 주도주 리스트 확인 (거래량 급증 + 상승률 상위)
3. 관심있는 종목 클릭 → 차트 뷰어로 이동
4. 이동평균선, RSI 등 기술적 지표로 분석
5. 매력적이라고 판단되면 관심종목에 등록
```

### 시나리오 2: 관심종목 모니터링
```
1. 사용자가 로그인 후 대시보드 접속
2. 관심종목 그룹별 시세 현황 확인
3. 특정 종목 클릭 → 차트 및 뉴스 확인
4. 새로운 뉴스가 있으면 내용 확인
5. 필요시 관심종목 메모 업데이트
```

### 시나리오 3: 뉴스 기반 투자 판단
```
1. 관심종목에 새 뉴스 알림 수신
2. 뉴스 피드에서 상세 내용 확인
3. 관련 종목 차트에서 가격 반응 확인
4. 추가 분석 후 매매 의사결정
```

### 시나리오 4: 매매 신호 설정 및 활용
```
1. 차트 뷰어에서 '매매 신호 설정' 진입
2. 프리셋 중 'RSI 과매도/과매수' 전략 선택 또는 커스텀 조건 생성
3. 조건 파라미터 조정 (예: RSI 기준값 30/70 → 25/75로 변경)
4. 신호 적용 후 차트에 과거 매수/매도 지점 표시 확인
5. 전략 저장 및 알림 설정
6. 이후 조건 충족 시 차트에서 신호 확인
```

## 5. 화면 구성

### 5.1 주요 페이지
| 페이지 | 설명 |
|--------|------|
| 대시보드 | 관심종목 요약, 주도주 하이라이트, 주요 뉴스 |
| 주도주 스크리너 | 모멘텀 지표 기반 주도주 리스트 |
| 종목 상세 | 차트 뷰어 + 종목 정보 + 뉴스 피드 |
| 관심종목 | 그룹별 관심종목 관리 |
| 설정 | 계정, 알림, 차트 기본값 설정 |

### 5.2 반응형 디자인
- 데스크톱: 풀 기능 제공 (다중 패널 레이아웃)
- 태블릿: 주요 기능 제공 (단일 패널)
- 모바일: 핵심 기능 제공 (시세 확인, 관심종목, 뉴스)

## 6. 비기능 요구사항

### 6.1 성능
- 차트 렌더링: 1초 이내
- 시세 업데이트: 실시간 (장중) / 1분 이내 (장 마감 후)
- 페이지 로딩: 2초 이내

### 6.2 보안
- 사용자 인증/인가 (회원가입, 로그인)
- 개인정보 암호화 저장
- HTTPS 통신

### 6.3 확장성
- 향후 모바일 앱 확장 고려
- 다양한 데이터 소스 연동 가능한 구조
- 신규 기술적 지표 추가 용이한 설계

### 6.4 가용성
- 장중(09:00-15:30) 99.9% 가용성 목표

## 7. 제약사항 및 의존성

### 7.1 데이터 소스
| 데이터 | 잠재적 소스 |
|--------|-------------|
| 주식 시세/차트 | 한국투자증권 OpenAPI, 키움증권 OpenAPI, KRX 정보데이터시스템 |
| 종목 뉴스 | 네이버 금융, 한경, 연합뉴스 등 |
| 재무 정보 | DART OpenAPI, 각 증권사 API |

### 7.2 제약사항
- 실시간 시세 제공 시 증권사 API 사용 조건 준수 필요
- 뉴스 크롤링 시 저작권 및 이용약관 준수 필요
- 투자 권유가 아닌 정보 제공 서비스임을 명시 필요 (면책조항)
