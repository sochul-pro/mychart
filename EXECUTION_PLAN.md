## Phase 0: 프로젝트 초기 설정

### TASK-001: 프로젝트 스캐폴딩
**설명**: Next.js 기반 프로젝트 생성 및 기본 구조 설정
**범위**:
- Next.js 14+ (App Router) 프로젝트 생성
- TypeScript 설정
- ESLint, Prettier 설정
- 폴더 구조 정의 (`/app`, `/components`, `/lib`, `/types` 등)
- Tailwind CSS 설정

**산출물**: 실행 가능한 빈 프로젝트
**의존성**: 없음

---

### TASK-002: 데이터베이스 설계 및 ORM 설정
**설명**: 사용자, 관심종목, 매매전략 등 데이터 모델 설계
**범위**:
- PostgreSQL + Prisma ORM 설정
- 스키마 설계:
  - User (사용자)
  - Watchlist (관심종목 그룹)
  - WatchlistItem (관심종목)
  - TradingStrategy (매매 전략)
  - SignalCondition (신호 조건)
- 마이그레이션 생성

**산출물**: Prisma 스키마, DB 마이그레이션
**의존성**: TASK-001

---

### TASK-003: 인증 시스템 구현
**설명**: 회원가입, 로그인, 세션 관리
**범위**:
- NextAuth.js 설정
- 이메일/비밀번호 인증
- JWT 세션 관리
- 로그인/회원가입 UI
- 보호된 라우트 미들웨어

**산출물**: 인증 플로우 완성
**의존성**: TASK-001, TASK-002

- 섹터 내 핫 종목 상위 5개 표시
- 반응형 그리드 레이아웃 (1~3열)

**핫 종목 점수 계산**:
| 지표 | 조건 | 점수 |
|------|------|------|

| 당일 등락률 | ≥10%/≥5%/≥3%/≥1%/<0% | +30/+20/+15/+10/-10 |
| 52주 신고가 | 돌파 시 | +20 |
| 52주 수익률 | ≥100%/≥50%/≥20%/≥0% | +20/+15/+10/+5 |

**산출물**:
- `/app/api/sectors/` - 섹터 API 라우트
